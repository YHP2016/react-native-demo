一.启动项目的方式

1.首先启动后台 切换到 server文件夹下面，安装node_module

npm install

然后启动 node app.js （如果有nodemon，直接热启动）

2.启动react-native项目， 进入ios目录，通过xcode打开nba_news.xcodeproj。

通过command+ R启动 app项目。
